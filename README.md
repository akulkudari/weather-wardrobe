Functioning Wardrobe and Weather website meant to find weather in user's area, and give them clothing suggestions based on the weather 
Uses MySQL remotely hosted database via Aiven to store user's data including session information for authentication, wardrobe items, devices including temperature and humidity sensors (via ESP32 and BMP180 temperature and pressure sensor)
All CRUD database operations are supported for wardrobe and device information. User authentication is done via ID, and all backend code for website is stored in app/main.py
